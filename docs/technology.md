# Technology Stack

## Frontend (Mobile)
- **Language:** Kotlin 2.x
- **UI Framework:** Compose Multiplatform (Android/iOS)
- **Networking:** Ktor Client (CIO/Darwin engines)
- **Serialization:** Kotlinx.serialization (JSON)
- **Dependency Injection:** Koin (Multiplatform)
- **Concurrency:** Kotlin Coroutines & Flow

## Backend
- **Framework:** Node.js NestJS
- **Database:** Document-based (Generic Repository Pattern for MongoDB/Firestore)
- **Auth:** Firebase Auth or JWT (to sync data across platforms)

## DevOps & Quality
- **CI/CD:** GitHub Actions (Automated builds, linting, and test execution)
- **Compatibility:** Android (API 24+), iOS (15.0+)
